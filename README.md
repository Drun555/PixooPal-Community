# PixooPal Community Clockfaces
This repository serves as a registry for third-party Clockfaces.

# I want to create a new clockface!!!

Woah, that's a courage! [It's easier than you think](https://github.com/Drun555/PixooPal-Preview).

## Pull Request Rules
If you are submitting a new clockface, do not change anything besides your new folder inside /src.
After PR, CI automatically validates and uploads the build.
